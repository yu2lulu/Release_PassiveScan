# Release_PassiveScan
###### 图形自动化安全测试工具


## 目前支持如下功能:
- 一键SQL注入扫描 ✅
- webpack文件泄漏检测 ✅
- XSS检测[Dom/Reflex] ✅
- 文件包含检测 ✅
- 网站目录结构生成 ✅
- 支持响应包敏感信息检测【身份证/手机号/护照/邮箱/银行卡】 ✅
- 数据包重放 ✅
- 支持敏感信息目录/文件 扫描 ✅
